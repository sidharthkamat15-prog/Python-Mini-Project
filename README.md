# 🐍 Python Mini Projects

A curated collection of **20 beginner-to-intermediate Python projects** focused on building strong fundamentals through real-world logic and hands-on practice.

Each project is designed to be **simple, practical, and explanation-driven**, helping improve problem-solving skills and code structure.

---

## 📂 Project Categories

### 📘 Basics
Fundamental programs to build core programming logic.

- **Calculator** → Performs basic arithmetic operations  
- **Word Counter** → Counts words in a given input  
- **Odd/Even Checker** → Determines number parity  
- **Temperature Converter** → Converts between Celsius, Fahrenheit, etc.  

---

### 🎮 Games
Fun CLI-based games to practice logic and user interaction.

- **Guess the Number** → Number guessing with hints  
- **Quiz Game** → Multiple-choice question system  
- **Tic Tac Toe** → Two-player grid-based game  
- **Dice Roller** → Simulates dice rolls  

---

### 🛠️ Utilities
Practical tools solving everyday problems.

- **Password Generator** → Generates secure random passwords  
- **To-Do List** → Manage tasks with add/remove features  
- **Contact Book** → Store and retrieve contacts  
- **Number Analyzer** → Analyze properties of numbers  
- **Password Strength Checker** → Evaluates password security  

---

### 💼 Applications
Mini real-world systems using structured logic.

- **ATM System** → CLI banking system (login, deposit, withdraw)  
- **Shopping Cart** → Simulates an online cart system  
- **Cafe Management** → Handles orders and billing  
- **Student Grade Manager** → Tracks and evaluates grades  
- **Log System** → Basic logging and tracking system  

---

## 🚀 Getting Started

Run any project using:

```bash
python filename.py

```
